Pie 饼图 ​

示例 ​

source 的基本使用 ​

和柱状图、折线图的source不同，饼图推荐对象数组的形式，即：每一项都是由name、value组成的对象

展开代码
vue
<template>
  <lc-pie
    style="width: 300px; height: 300px"
    :source="source"
  />
</template>
<script>
export default {
  data() {
    return {
      source: [
        { name: '分类一', value: 1048 },
        { name: '分类二', value: 735 },
        { name: '分类三', value: 580 },
      ],
    }
  },
}
</script>

label 在一行展示完全 ​

echarts 默认radius外半径是75%，比较大，当 label 字数比较多且宽高较小时，容易出现省略号，甚至溢出的情况。

此时可以手动调整radius的大小，来让 label 展示完全。此处我们设置成46%。

展开代码
vue
<template>
  <lc-pie
    style="width: 300px; height: 300px"
    :source="source"
    :radius="[0, '46%']"
  />
</template>
<script>
export default {
  data() {
    return {
      source: [
        { name: '分类一', value: 1048 },
        { name: '分类二', value: 735 },
        { name: '分类三', value: 580 },
      ],
    }
  },
}
</script>

圆环图 ​

同样，可以调整radius的内半径，使其成为一个圆环图

展开代码
vue
<template>
  <lc-pie
    style="width: 300px; height: 300px"
    :source="source"
    :radius="['25%', '46%']"
  />
</template>
<script>
export default {
  data() {
    return {
      source: [
        { name: '分类一', value: 1048 },
        { name: '分类二', value: 735 },
        { name: '分类三', value: 580 },
      ],
    }
  },
}
</script>

有时，业务需要在内环里额外展示一些固定数据，我们最先想到的可能是把label.position设置成center。但这样默认的outside标签效果就没有了，而且这个center的效果是鼠标移入某一块才会在中心区域展示该块标签，并不符合“展示固定数据”的需求。

如果我们既想要默认的outsidelabel 效果，又想要居中固定的文字，可以通过graphic来绘制。详细配置可以参考：ECharts graphic 文档

展开代码
vue
<template>
  <lc-pie
    style="width: 300px; height: 300px"
    :source="source"
    :radius="['25%', '46%']"
    :graphic="graphic"
  />
</template>
<script>
export default {
  data() {
    return {
      source: [
        { name: '分类一', value: 1048 },
        { name: '分类二', value: 735 },
        { name: '分类三', value: 580 },
      ],
      graphic: {
        elements: [
          {
            type: 'group',
            left: 'center',
            top: 'center',
            children: [
              {
                type: 'text',
                style: {
                  text: 'code',
                  fill: 'gray',
                  fontSize: 12,
                },
                top: -10,
                left: 'center',
              },
              {
                type: 'text',
                style: {
                  text: '1024',
                  fill: 'blue',
                  // v5.5 尝试文档中的font简写不生效，只能一个属性一个属性写
                  fontSize: 16,
                  fontWeight: 'bold',
                },
                top: 10,
                left: 'center',
              },
            ],
          },
        ],
      },
    }
  },
}
</script>
